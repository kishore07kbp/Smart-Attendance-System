# Smart Attendance System

Smart Attendance System using MERN Stack, Flutter, and IoT (ESP32 + MQTT).

## Technology Stack

| Category | Technologies |
|-----------|-------------|
| Frontend | React.js, HTML5, Vanilla CSS |
| Backend | Node.js, Express.js |
| Database | MongoDB Atlas |
| Mobile App | Flutter |
| IoT | ESP32 |
| Communication | MQTT, Bluetooth Low Energy (BLE) |
| Tools | VS Code, Git, GitHub, Postman, MongoDB Compass |

## Features
- Live Face Scanning for student identity verification.
- Current Course-Based Attendance marking for active classes.
- Device Binding mechanism to prevent proxy attendance.
- Flutter-based mobile application (Bunk Tracer) for attendance management.
- Bluetooth Low Energy (BLE) advertisement broadcasting with unique permanent ID and roll number.
- ESP32 IoT nodes for receiving BLE advertisement packets.
- MQTT-based communication for real-time data transmission to the server.
- Real-time attendance tracking and monitoring.
- Automated attendance recording and storage.
- Attendance history and student-wise attendance reports.
- Faculty dashboard for attendance management and analytics.
- Course and timetable management.
- Low-cost, scalable IoT-based attendance infrastructure.
- Reduced manual effort and improved attendance accuracy.
